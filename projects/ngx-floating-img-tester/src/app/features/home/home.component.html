ngx-floating-img

Mobile look and feel image viewer for Angular 6+

installation

  • Add ngxfloatingimg to your Angular project

    npm install --save ngx-floating-img

  • Once installed import NgxFloatingImgModule in your app.module.ts

    NgxFloatingImgModule.forRoot()

  • Add ngx-floating-img in your template file

    <ngx-floating-img></ngx-floating-img>

This example shows multiple options can be applied to ngx-floating-img component, see all examples

Tested on Chrome, Firefox, Safari, IE10+, Edge
